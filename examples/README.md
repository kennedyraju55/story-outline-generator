# Examples for Story Outline Generator

This directory contains example scripts demonstrating how to use this project.

## Quick Demo

```bash
python examples/demo.py
```

## What the Demo Shows

- **`load_config()`** — Call load_config
- **`get_character_archetypes()`** — Call get_character_archetypes
- **`get_plot_structures()`** — Call get_plot_structures
- **`get_worldbuilding_categories()`** — Call get_worldbuilding_categories
- **`build_prompt()`** — Build the story outline generation prompt.

## Prerequisites

- Python 3.10+
- Ollama running with Gemma 4 model
- Project dependencies installed (`pip install -e .`)

## Running

From the project root directory:

```bash
# Install the project in development mode
pip install -e .

# Run the demo
python examples/demo.py
```
